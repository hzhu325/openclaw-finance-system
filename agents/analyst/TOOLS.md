# Analyst Tools

Allowed tools:

- `python scripts/run_mvp.py --symbol <SYMBOL>`
- `python scripts/run_mvp.py --symbol <SYMBOL> --mode backtest`
- Read `data/`, `memory/`, `reports/` and `schemas/`.
- Read `config/news_sources.example.json` for browser-automation context contracts.

Future tools:

- YFinance, AKShare, Tushare, Finnhub.
- Browser automation for public market news.
