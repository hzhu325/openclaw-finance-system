# Risk Tools

Allowed tools:

- `python scripts/run_mvp.py --symbol <SYMBOL>`
- Read `reports/latest_signal.json`.
- Read `reports/latest_risk_review.json`.
- Read `reports/latest_human_approval_card.json`.
- Read `schemas/trade_signal.schema.json`.

Forbidden tools:

- Live broker APIs.
- Any command that changes external positions.
