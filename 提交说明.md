# 提交说明

本文件用于说明项目材料的组成和运行方式。

## 1. 提交内容

本次项目材料打包为：

```text
OpenClaw_finance_submission.zip
```

其中主要包括：

- `openclaw-finance-system/`：项目源码和配置文件
- `OpenClaw_multi_agent_finance_report.docx`：项目实现报告
- `docs/`：架构说明、智能体设计、运行手册和补充文档
- `reports/`：最新一次运行生成的信号、风控、审批和回测结果
- `dashboard.html`：本地展示页面
- `open_dashboard.bat`：一键运行和展示脚本

## 2. 运行方式

解压后进入 `openclaw-finance-system` 目录，双击：

```text
open_dashboard.bat
```

脚本会自动生成最新运行结果、执行测试，并在浏览器中打开展示页面。

## 3. 验证要点

展示页面中可以重点查看以下内容：

- 交易标的显示为 `TEST`
- 交易建议显示为 `SELL`
- 风控结论显示为 `APPROVE_WITH_HUMAN_CONFIRMATION`
- HITL 状态显示为 `PENDING_HUMAN_APPROVAL`
- `execution_released=false`，表示系统不会真实下单
- 页面下方包含多空三轮辩论、核心指标和回测结果

命令行窗口中出现以下结果，表示自动测试通过：

```text
Ran 3 tests
OK
```

## 4. 项目说明

本项目是一个 simulation-only 的多智能体金融交易系统原型，主要用于展示行情处理、指标计算、多智能体辩论、结构化交易信号、风控审查、HITL 人工审批和回测展示流程。

当前版本不连接真实券商账户，不执行真实交易，也不构成投资建议。
